🚀 How to Run This Project Locally

Follow these steps:

1. Clone the repository

git clone https://github.com/your_user_name/gshivani_portfolio.git

2. Go to the project folder
   
cd gshivani_portfolio

3. Install dependencies

npm install

4. Run the project

npm run dev

5. Open in browser
   
Go to: 
http://localhost:5173
